# Image as Numbers Analyzer

## 📌 Description
A computer vision mini-project that demonstrates how images are stored as numerical matrices.

## 🚀 Features
- Loads an RGB image
- Prints image shape
- Shows pixel values
- Splits RGB channels
- Visualizes each channel

## 🛠 Technologies
- Python
- OpenCV
- Matplotlib

## ▶ Run
pip install -r requirements.txt
python app.py
